# WAKPOO

집중 시간과 할 일을 관리하고, 팀원과 진행 상황을 공유하는 모바일 웹 프로토타입입니다.

## 주요 기능

- 할 일 등록 및 완료
- 25분, 50분, 90분 집중 타이머
- 팀 생성, 초대, 팀 할 일과 진행률 관리
- 마이 페이지, 알림, 기록 화면

## 실행 방법

Node.js 22.13 이상이 필요합니다.

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 열면 됩니다.

## 빌드

```bash
npm run build
```
